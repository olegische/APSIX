# APSIX

Agent Portable Semantic Interface eXtension

Minimal primitive contract for portable semantic runtimes.

Internet-Draft candidate: `draft-romanchuk-apsix-00`

Author: Oleg Romanchuk  
Date: March 10, 2026  
Status: Research Specification

This repository publishes the APSIX runtime specification.

APSIX defines a minimal primitive contract for governed semantic runtimes,
including zone lifecycle, spawn governance, execution semantics, artifact
anchoring, replay requirements, and membrane-mediated expansion control.

The `draft-romanchuk-apsix-00` package is intended to serve as a public,
timestamped specification snapshot. Submission as an IETF Internet-Draft is
planned when the submission window reopens.

## Repository Scope

This repository contains the APSIX Draft 00 specification snapshot prepared for
public publication. The intended canonical public snapshot is the Git tag and
GitHub Release named `apsix-draft-00-2026-03-10`.

Draft 00 is a fixed publication snapshot. Subsequent technical changes should
be published as a new draft revision rather than modifying the released Draft
00 specification in place.
